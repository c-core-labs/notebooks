Jupyter notebooks for demonstrating helpful things


## Use with Poetry

### Install dependencies

``` bash
poetry install
```

### Start notebook server
``` bash
poetry run start
```

### Update requirements.txt dependencies
``` bash
poetry run freeze
```


## Use without Poetry (plain old pip)


### Install dependencies
``` bash
pip install -f requirements.txt
```

### Run notebook server

``` bash
jupyter lab
```
